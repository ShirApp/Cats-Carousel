# Cat Facts Carousel :)

The app includes a carousel of cat cards, in which you can save your favorite ones!
Are you ready for the cool animation? :)

## Installing

- `npm install`
- `npm start`

## Built With

- [Create React App](https://reactjs.org/docs/create-a-new-react-app.html)
- [React Router](https://reacttraining.com/react-router/)
- [Redux](https://redux.js.org/)
- [React Card Carousel](https://www.npmjs.com/package/react-card-carousel)
- [Semantic-UI](https://react.semantic-ui.com/)

## APIs

- [The Cat API](https://thecatapi.com/)
- [Cat Facts API](https://alexwohlbruck.github.io/cat-facts/docs/)
